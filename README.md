# Iris Species Prediction


Project based on dataset from kaggle   
https://www.kaggle.com/datasets/uciml/iris

### Techniques used in the project

+ Visualization:
    + correlation heatmap

+ Logaritmic Regression
+ NaiveBias
    + Gaussian NaiveBias
    + Categorical NaiveBias
+ Decision tree
+ Grid search
+ Metrics
    + f1_score
    + precison_score
    + recall_score
    + accuracy_score
